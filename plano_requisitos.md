# Plano de Requisitos - Demanda+

## Requisitos funcionais
- Cadastro de produtos.
- Entrada de demandas históricas.
- Previsão multissemanal.
- Métodos: ingênuo, média móvel simples, média móvel ponderada, suavização exponencial e regressão linear.
- Comparação automática de modelos.
- Gráficos, tabelas, KPIs e análise de erros.
- Recomendação gerencial.
- Estoque, capacidade, reposição e cenários what-if.
- Exportação de relatórios.

## Requisitos não funcionais
- Interface em português.
- Layout legível e responsivo.
- Código organizado e comentado.
- Não exigir banco de dados, login ou API externa.
- Tratamento de erros para poucos dados, vazios e valores negativos.
- Transparência: previsão não é certeza, é apoio à decisão.
