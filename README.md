## 🚀 Sobre o Projeto

O **Demanda+** é um aplicativo web desenvolvido em **Python + Streamlit** com foco em previsão de demanda, análise operacional e apoio à tomada de decisão gerencial.

O sistema transforma dados históricos de vendas e produção em previsões inteligentes, indicadores estratégicos e recomendações operacionais para pequenas empresas.

---

# 🎯 Objetivo

Auxiliar empresas no:

- planejamento da produção;
- controle de estoque;
- redução de desperdícios;
- previsão de demanda;
- apoio à tomada de decisão;
- monitoramento operacional.
